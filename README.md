# GPS Cruise Tour Audio Guide (No WiFi)

GPS 기반 크루즈 투어 오디오 가이드 애플리케이션입니다. 와이파이 없이도 사용 가능합니다.

## 📍 Repository

- **GitHub URL**: https://github.com/maibauntourph-collab/GPS-Cruise-Tour-AudioGuideNo-Wifi

## 🚀 주요 기능

- GPS 기반 위치 추적
- 오프라인 오디오 가이드
- 다국어 지원 (한국어, 영어, 중국어, 일본어, 프랑스어, 스페인어, 독일어, 이탈리아어, 포르투갈어, 러시아어)
- 크루즈 투어 경로 안내
- 랜드마크 정보 제공

## 🛠️ 기술 스택

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Backend**: Node.js
- **Audio**: Clova TTS

## 📁 프로젝트 구조

```
├── client/          # 프론트엔드 소스 코드
├── server/          # 백엔드 소스 코드
├── public/          # 정적 파일 (오디오 등)
├── shared/          # 공유 스키마
└── scripts/         # 유틸리티 스크립트
```

## 🏃 실행 방법

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```
